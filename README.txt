Игра-симулятор фермера "Farm Simulator"

Техническое задание:
    Реализующиеся окна:
        Стартовое
        Окно уровней
        Игровое
        Окно магазина
    Уровни:
        Лёгкий
        Продвинутый
        Сложный
        Свободный режим

Использующиеся инструменты:
    IDE PyCharm
    VCS Git
    Python

Пояснительная записка:
    images - папка с изображениями
    sounds - папка со звуками
    fonts - папка со шрифтом
    main.py - файл с итоговым продуктом
    остальные файлы - части проекта, отвечающий за некоторые области функционала

структура приложения:
    начать игру - кнопка, открываюшая меню с выбором сложности(лëгкий, продвинутый, сложный, свободная игра). по нажатии на выбор сложности открывается меню с игрой
    продолжить игры - открывается последнее сохранение свободной игры(кнопка отображается только при наличии файла save.json в локальной директории) 
    титры - титры игры
    результаты - меню для просмотра рейтинга по всем сложностям за исключением свободной игры
    выйти - выход из игры

структура игрового меню:
    хотбар внизу экрана, по центру - поле для высадки овощей, сверху справа - кнопка для открытия магазина, а сверху слева - кнопка паузы(пауза также вызывается клавишей escape
    меню паузы:
        кнопка выйти - выход на стартовый экран, в свободном режиме сохраняется прогресс
        кнопка вернуттся - пауза снимается
        кнопка заново - начать заново игру с той же сложностью
    меню магазина:
       6 кнопок с различными товарами(3 овоща, ведро воды и 2 улучшения) 

игровой процесс:
    игроку необходимо собрать необходимое количество монет как можно быстрее(в свободном режиме нет этого правила) 
    игроку на старте доступны семена картофеля и ведро с водой
    игрок может покупать семена, воду и улучшения в магазине
    игроку необходимо разместить семена из хотбара на поле, удобрить их водой и ждать их выраста, после собирать монеты
    если во время не собрать овощ, то клетка перестанет быть доступной
